# ВКР Бакалавр ПИ

### Тема: Сервис управления предметно-ориентированными знаниями с использованием гетерогенных источников

 * Язык программирования: Go
 * Архитектура: Клиент-серверная
 * Backend API: [gin](https://github.com/gin-gonic/gin)
 * Frontend TUI: [bubbletea](https://github.com/charmbracelet/bubbletea)

| Функция | Описание |
| ------- | ----------- |
| Аутентификация | При помощи API Key |
| Добавление записей | Текстовые записи, файлы PDF, содержимое WEB-страниц по URL |
| Тегирование | Добавление тегов к записям, автотеггинг |
| Поиск | Поиск по названию записи, по содержимому, по тегам, фильтрация, пагинация |
| Просмотр | Просмотр и редактирование содержимого записей |
| Экспорт | Экспорт библиотеки записей и тегов в JSON |
